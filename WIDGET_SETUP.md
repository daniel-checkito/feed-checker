# Motivational Productivity Widget Setup

This is a custom Scriptable widget designed to keep you motivated and productive throughout the day.

## Features

✨ **What the widget includes:**
- Personalized greeting (Good Morning/Afternoon/Evening)
- Current date
- Random motivational quotes from successful people
- Daily productivity tips and reminders
- Current time display

## Installation

### Step 1: Download Scriptable App
1. Open the App Store on your iPhone
2. Search for "Scriptable" (by Simon Strandgaard)
3. Install the app

### Step 2: Add the Widget Script
1. Copy the code from `motivational-widget.js`
2. Open the Scriptable app
3. Tap the "+" button to create a new script
4. Paste the entire code into the editor
5. Tap "Done"
6. Name it "Motivational Widget" (or your preferred name)

### Step 3: Add to Home Screen
1. Go to your iPhone home screen
2. Long-press on an empty area to enter edit mode
3. Tap the "+" button in the top-left corner
4. Search for "Scriptable"
5. Select it and choose a widget size (Medium or Large recommended)
6. Select your "Motivational Widget" script
7. Tap "Add Widget"

### Step 4: Set Up Auto-Refresh (Optional)
For the widget to refresh automatically every 5 minutes:
1. In Scriptable, open your widget script
2. The refresh timing is already configured in the script

## Customization

You can easily customize the widget:

### Change Colors
Edit these lines to use your favorite colors:
- `"#1a1a2e"` - Dark blue background
- `"#f1faee"` - Off-white text
- `"#a8dadc"` - Light cyan accents
- `"#e63946"` - Red time display

### Add Your Own Quotes
1. Open the script in Scriptable
2. Find the `quotes` array
3. Add your favorite quotes in the format: `"Your quote - Author name"`

### Add Your Own Tips
1. Find the `productivityTips` array
2. Add new productivity reminders with emoji

## Widget Sizes

- **Small Widget**: Shows greeting + quote (1x1 grid)
- **Medium Widget**: Full experience with all features (2x1 grid)
- **Large Widget**: Extra spacious version (2x2 grid)

## Troubleshooting

**Widget not updating?**
- Make sure location access is enabled for Scriptable
- Try removing and re-adding the widget
- Restart your phone

**Can't see the script in Scriptable?**
- Make sure you copied the entire code
- Check that the script was saved (look for the name at the top)

**Want different quotes?**
- Edit the quotes array in the script
- The widget picks a random quote each time it updates
